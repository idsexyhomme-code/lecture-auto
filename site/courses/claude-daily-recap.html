Claude로 5분 하루 회고 자동화

매일 5분으로 1인 사업가의 회고를 자동 완성한다

결과 약속

  • Claude 프롬프트 하나로 하루 회고 초안을 5분 안에 생성할 수 있다
  • 나만의 회고 템플릿을 설계하고 Claude에 저장·재사용할 수 있다
  • 일주일치 회고 데이터를 Claude로 분석해 사업 인사이트 요약본을 만들 수 있다

타깃: 매일 일기·회고 작성에 부담을 느끼는 1인 사업가

선수지식: Claude.ai 계정 보유 및 기본 채팅 사용 경험, 회고 또는 일기를 작성해본 경험 (형식 무관)

차시별 커리큘럼

# 제목 학습목표 실습 분
1 5분 회고 프롬프트 설계하기 오늘 하루의 업무·감정·결정 정보를 입력하면 회고 초안을 즉시 출력하는 Claude 프롬프트를 1개 작성한다 나의 하루 데이터를 넣어 회고 초안을 생성하는 마스터 프롬프트 문서 (Notion 또는 메모장 1페이지) 15
2 나만의 회고 템플릿 고정하기 매일 동일한 구조로 회고가 출력되도록 커스텀 템플릿 변수(섹션·질문 항목)를 정의하고 Claude 시스템 프롬프트에 적용한다 섹션 3개 이상을 포함한 나만의 회고 템플릿이 적용된 Claude 시스템 프롬프트 파일 (복사·붙여넣기 즉시 사용 가능한 텍스트 파일) 15
3 일주일 회고를 인사이트 요약본으로 변환하기 7일치 회고 텍스트를 Claude에 입력해 반복 패턴·개선 우선순위·다음 주 액션 아이템을 포함한 주간 인사이트 요약본을 생성한다 7일치 회고를 분석한 주간 인사이트 요약 PDF (Claude 출력 → 복사 → 문서화 완성본) 15

평가

수강자가 직접 제작한 마스터 프롬프트·커스텀 템플릿·주간 인사이트 요약본 3종을 영상으로 화면 녹화하여 제출하면 완주 인증 발급
